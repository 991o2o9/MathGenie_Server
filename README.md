# MathGenie — образовательная платформа на Express + MongoDB

## Запуск

1. Установить зависимости:
   ```
   npm install
   ```
2. Создать файл `.env` на основе `.env.example` и заполнить переменные.
3. Запустить сервер:
   ```
   npm run dev
   ```

## Основные технологии

- Express.js
- MongoDB + Mongoose
- JWT
- AdminJS
- Swagger
- HuggingFace API
